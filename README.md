# MendingLevels
A Minecraft fabric mod that adds enchantment level support for the mending enchantment

## Downloads
- Modrinth: https://modrinth.com/mod/mendinglevels
- CurseForge: https://www.curseforge.com/minecraft/mc-mods/mending-levels
